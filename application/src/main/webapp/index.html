Welcome to Your Application

This is your SAP Cloud Platform Cloud Foundry TomEE application powered by the SAP S/4HANA Cloud SDK.

  • To get started, visit the HelloWorldServlet. You can find its source code file in application/src/main/java/.
  • To run this application locally, execute mvn package tomee:run within the application directory.
  • Additional additional static content can be placed inside application/src/main/webapp/.

For more information and tutorials, visit https://sap.com/s4sdk.
